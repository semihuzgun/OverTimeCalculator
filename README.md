# OverTimeCalculator

A Flutter application for calculating employee overtime and working hours quickly and accurately.

## Features

* Overtime calculation
* Daily and monthly work-hour tracking
* User-friendly interface
* Instant calculation results
* Cross-platform support (Android, iOS, Web)

## Getting Started

Clone the repository:

```bash
git clone https://github.com/semihuzgun/OverTimeCalculator.git
```

Navigate to the project folder:

```bash
cd OverTimeCalculator
```

Install dependencies:

```bash
flutter pub get
```

Run the application:

```bash
flutter run
```
